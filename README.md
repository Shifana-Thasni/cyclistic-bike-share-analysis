# 🚲 Cyclistic Bike-Share Analysis

## 📋 Business Task
Analyze how annual members and casual riders use Cyclistic bikes 
differently and provide recommendations to increase memberships.

## 🔧 Tools Used
- **Python** (Pandas, NumPy) — data cleaning and analysis
- **Tableau** — interactive dashboard and visualizations
- **Jupyter Notebook** — analysis environment

## 📂 Data Source
Divvy Bike Share trip data:
- Divvy_Trips_2019_Q1.csv
- Divvy_Trips_2020_Q1.csv

## 🔍 Analysis Process
1. Data Loading
2. Data Cleaning
3. Data Transformation
4. Exploratory Data Analysis
5. Visualization

## 📊 Key Findings
- Members ride more frequently than casual riders
- Casual riders tend to have longer ride durations
- Ride patterns vary significantly by day of the week

## 💡 Recommendations
- Create targeted marketing campaigns to convert casual 
  riders into annual members
- Offer weekend membership incentives since casual riders 
  ride more on weekends
- Highlight cost savings of annual membership for 
  long-duration riders
